This is Ruby on Rails project Book Shop Review

This is a Ruby on Rails, HTML, Bootstrap project, was created to review books. With the ability to add new books, add a book cover, indicate the name of the book, the name of the author, a summary. Books are added by category, with the ability to edit and delete by the author of the post. To create a post, you must register. For review, registration is optional.

This is Ruby on Rails project New Tasklist

These are Ruby on Rails, HTML, Bootstrap, Tasklist. With the ability to create tasks for every day. By specifying the task name, creation time, and author name. Ability to edit and delete tasks. by the author. View the time and success of the assignment.

This is E-Commerce project on React

    • E-commerce, JS, React, HTML, a shop for online trading in smartphones. It was made for educational purposes with JS React. This is a functional application, although it was developed for educational purposes. Everything works, indicating the price, displaying the amount of selected goods in the customer’s basket and automatically calculating the total amount. The PayPal payment module is disabled, because the project was done for educational purposes with React JS. The application operates on the Internet, and clicking on the first screen opens the page with the application.
